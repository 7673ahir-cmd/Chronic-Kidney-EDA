# 🩺 Chronic Kidney Disease EDA

Exploratory Data Analysis (EDA) on the Chronic Kidney Disease dataset (400 rows × 25 columns) using Python.  
This project focuses on cleaning, visualizing, and analyzing medical data to uncover insights about CKD risk factors.

---

## 📊 Project Highlights
- Data cleaning (handling missing values, fixing categorical inconsistencies)
- Feature engineering (numeric conversion of medical attributes)
- Univariate, bivariate, and multivariate analysis
- Visualizations:
  - Histograms, boxplots, violin plots
  - Count plots and pie charts
  - Pairplots and correlation heatmaps
- Medical insights:
  - CKD patients show higher **blood urea** and **serum creatinine**
  - Lower **haemoglobin** levels strongly linked with CKD
  - Hypertension and diabetes are common comorbidities

---

## ⚙️ Tech Stack
- **Python**  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Plotly
